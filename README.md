# Space Maze

Space Maze is a 3D browser game built with Astro, TypeScript, and Three.js. The project explores lightweight web-based 3D presentation through a maze-like space setting.

## Highlights

- Three.js-powered space environment
- Astro-based project structure and deployment
- Responsive presentation for desktop and mobile

## Quick start

```bash
bun install
bun run dev
```

Build and preview:

```bash
bun run check
bun run build
```
